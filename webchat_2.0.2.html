The first ever portable Sinhala-English code-switching keyboard interface.

  • Can be attached to any website.
  • Fully written in Vanilla JavaScript.
  • Seamlessly integratable with rasa web-chat.
  • Highly customizable.

Instructions:

  1. Click Chat-Bot icon in Right-Bottom corner.
  2. Then, Chat-Bot Coversation UI will open.
  3. Type anything you want to know from the Chat-Bot.
  4. Press enter. Then Chat-Bot will respond with the related subject.

Note: By Default the keyboard is allowed only Sinhala typing. You can use CTRL+Q to switch into English.
